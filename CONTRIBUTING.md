# Contributing

Thanks for your interest in Navo.

## Local Setup

```bash
pnpm install
pnpm dev:chrome
```

## Before Opening A Pull Request

```bash
pnpm typecheck
pnpm build:chrome
```

## Guidelines

- Keep V1 local-first and privacy-friendly.
- Do not request permissions beyond the feature scope.
- Do not commit secrets, browser profile data, `node_modules`, or build output.
- Use Conventional Commits for commit messages.
